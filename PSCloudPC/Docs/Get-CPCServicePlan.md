# Get-CPCServicePlan

## SYNOPSIS
This function will return all currently available service plans

## SYNTAX

```
Get-CPCServicePlan [-ServicePlanName <String>] [<CommonParameters>]
```

## DESCRIPTION
This function will return all currently available service plans

## EXAMPLES

### EXAMPLE 1
```
Get-CPCServicePlans
```

### EXAMPLE 2
```
Get-CPCServicePlans -ServicePlanType "enterprise"
```
